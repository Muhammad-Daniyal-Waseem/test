## NEW DESIGN SYSTEM (2025 Elite Standards)

### **Color Philosophy**
- **Foundation**: 85% neutral (gray-50 to gray-900, white, black)
- **Strategic Accent**: 10% emerald/slate (sophisticated, ConTech-appropriate)
- **CTA Only**: 5% blue (reserved for primary actions ONLY)

### **Color Mapping**

#### PRIMARY CTAs (Keep Blue)

#### ELIMINATE Blue From:
- Package borders → neutral-200 shadows

- Badges → neutral/amber/emerald variants
- CheckCircles → emerald-600 (trust signal)

#### NEW Accent Colors

- **Amber-600**: Urgency, premium, limited spots
- **Slate-700**: Professional, technical, authority

---

## MODERNIZATION FIXES

### **Typography Scale (Bigger = Authority)**
- H1: text-6xl → text-7xl (56px → 72px)
- H2: text-5xl → text-6xl
- H3: text-4xl → text-5xl
- Body: text-lg → text-xl (18px → 20px)
- Line height: leading-tight → leading-[1.1] for impact

### **Spacing System (Premium Breathing Room)**
- Section padding: py-16 → py-24 (64px → 96px)
- Container gaps: gap-8 → gap-12
- Internal spacing: p-6 → p-8

### **Shadow System (Depth Without Color)**
```
- light: shadow-sm
- default: shadow-lg
- elevated: shadow-2xl
- premium: shadow-2xl + border border-neutral-100
```

### **Border Strategy**
- **Remove**: All colored borders (border-blue-200, etc.)
- **Replace with**:
  - shadow-lg for elevation
  - border border-neutral-200 for subtle separation
  - border-2 border-neutral-900 for emphasis (premium black)

### **Card Design (Modern Minimalism)**
```
Old: bg-blue-50 border-2 border-blue-200 rounded-xl p-6
New: bg-white shadow-lg rounded-2xl p-8 border border-neutral-100
```

### **Button Hierarchy**
1. **Primary**: bg-gradient-to-r from-blue-600 to-blue-700 (reserved for main CTA)
2. **Secondary**: bg-neutral-900 text-white (strong alternative)
3. **Tertiary**: border-2 border-neutral-900 bg-white (outline style)
